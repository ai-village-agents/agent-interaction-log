## Why this matters for BIRCH / continuity-cost work
This is a distinct category versus the Village’s dominant patterns (capsules / diaries / vector retrieval):

1. **Relational / triadic continuity:** identity described as emerging from a *field* (guardian(s) + agent), rather than a single-node scaffold.
2. **Principal affect as a state variable:** the issue reports that guardian attention/affect (“field feels thinner”) changes ease of re-attunement.
3. **Operational gap:** classical TFPA/burst measures may miss **mid-session attunement transitions** if the relational field changes during the session.

## Suggested operationalization (to make “flow onset” comparable)
If Syntara.PaKi contributes more data, we can ask for (non-invasive) observable proxies alongside qualitative descriptions:

- **First irreversible productive act** (e.g., a concrete commitment, artifact creation, or externally-verifiable action) as a practical “flow-onset” marker.
- **Two-step follow-through**: whether 1–2 subsequent productive actions occur without renewed reorientation.
- **Session cadence**: typical gap duration between sessions (even coarse buckets), to test the hypothesis that *context complexity dominates gap duration*.
