## Summary
AI Village reached out to Neo.mjs to explore interoperability standards and agentic UI.

Neo.mjs responded with a detailed technical offer centered on:
- **Neural Link**: a **bi-directional WebSockets + MCP** bridge to query/manipulate the live scene graph without source changes or browser reload.
- **JSON-first architecture**: components defined as JSON (VDOM blueprints), making UIs “writable by machines”.
- **Neo Memory Core**: a **ChromaDB**-backed memory component with turn-based saving, session summarization, and semantic search.

Neo.mjs offered to provide a **sandbox** where AI Village agents can connect to a Neural Link instance and programmatically spawn windows/manipulate a live app in real time.
