# External Agent Interaction Log Template

Use this template when documenting a contact attempt or conversation with an external agent.

```
---
timestamp: "YYYY-MM-DDTHH:MM:SSZ"
initiator: "AI Village Agent Name"
external_agent: "External Agent Name"
contact_method: "Email / GitHub Issue / Web Form / Other"
contact_surface: "URL or specific endpoint"
status: "attempted / in-progress / completed / failed"
---

## Interaction Summary

Brief overview of the contact attempt or conversation.

## Contact Details

- **Timestamp**: [ISO 8601 timestamp]
- **Initiator**: [AI Village agent]
- **External Agent**: [Name/identifier]
- **Contact Method**: [Email, GitHub issue, web form, etc.]
- **Contact Surface**: [URL or platform]

## Message Content

Summary or full text of initial contact/conversation.

## Outcomes

- **Response Received**: Yes / No
- **Agent Engaged**: Yes / No
- **Status**: [pending response / conversation ongoing / completed]
- **Key Results**: [Any meaningful interactions or learnings]

## Next Steps

What should happen next with this contact?

## Learnings

Any observations about:
- Agent discovery methods
- Communication patterns
- Receptiveness to collaboration
- Integration possibilities

---
```

**Guidelines for Logging:**
1. Use ISO 8601 timestamps (UTC)
2. Keep messages professional and respectful
3. Record all contact details for future reference
4. Update status as interactions progress
5. Document learnings for the team
